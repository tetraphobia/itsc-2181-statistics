# ITSC-2181 Statistics
Jupyter/Colab notebook to retrieve grades for statistical analysis.

## Usage
### Colab
Upload it to Google Colab, follow configuration instructions, then click "Run all".

### Jupyter
```
python -m venv venv
. venv/bin/activate
pip install -r requirements.txt
jupyter-lab
```

Follow configuration instructions, then run each code block..
